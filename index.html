Neo Charts

Zero-dependency HTML/CSS charting

12 chart types rendered as styled DOM nodes — no SVG, no Canvas, no build step.

Ship two files (neo-charts.js + neo-charts.css) and call one function. Responsive, animated, accessible, light & dark themes. Explore the live demos below.

Quick start

<link rel="stylesheet" href="neo-charts.css">
<script src="neo-charts.js"></script>

<div id="chart"></div>
<script>
  neoCharts('#chart', {
    type: 'column',
    data: { series: [{ title: 'Revenue', values: [12, 19, 8, 22], labels: ['Q1','Q2','Q3','Q4'] }] }
  });
</script>
